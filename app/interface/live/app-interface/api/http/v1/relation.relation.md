## [app端关注二级页][全量]正在直播接口

`GET http://api.live.bilibili.com/xlive/app-interface/v1/relation/liveAnchor`

### 请求参数

|参数名|必选|类型|描述|
|:---|:---|:---|:---|
|buyaofangqizhiliao|否|string| 调试咒语|
|platform|否|string| 平台|
|device|否|string| 设备|
|build|否|string| 版本号|
|sortRule|否|integer| 排序类型|
|filterRule|否|integer| 筛选类型|
|quality|否|integer| 清晰度|
