#### business/blademaster

> Out of Box Middleware

##### 项目简介

来自 bilibili 主站技术部的 blademaster middleware，目前以下 middleware 已经 Ready for Production：

- Antispam
- Limiter
- Supervisor
- Degrade

##### 项目特点

- 模块化设计，一个模块只干一件事

##### 编译环境

- **请只用 Golang v1.8.x 以上版本编译执行**

##### 依赖包

###### Limiter:

- [x/time/rate](golang.org/x/time/rate)
